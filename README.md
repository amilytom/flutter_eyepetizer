## 项目结构

``` bash
├─components                 // 公共的一些组件
│
├─fijkplayer_skin            // fijkplayer皮肤
│
├─schema                     // json解析类，模型
│
├─utils                      // 工具函数
│
├─pages                      // 所有的页面（前端思维）
│
├─main.dart                  // 主入口文件
│
├─widget                     // 一些小部件
│
├─service                    // getx 全局控制器
│
├─request                    // dio亲妈级别封装
│
├─router                     // 路由系统
```
